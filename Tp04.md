## Respuestas teóricas

### ¿Por qué el perceptrón no puede resolver XOR?

El perceptrón no puede resolver el problema XOR debido a que este no es linealmente separable.

Esto implica que no existe una única recta que permita separar correctamente las clases. En el caso del problema XOR, los puntos se encuentran distribuidos de forma tal que no pueden dividirse mediante una frontera lineal.

Dado que el perceptrón solo puede aprender fronteras de decisión lineales, no es capaz de encontrar una solución adecuada para este tipo de problema.

---

### ¿Qué sucede al modificar el learning rate?

El learning rate, o tasa de aprendizaje, determina la magnitud de los ajustes realizados sobre los pesos durante el proceso de entrenamiento.

- Un learning rate bajo produce un aprendizaje lento y puede requerir una mayor cantidad de épocas para converger.
- Un learning rate adecuado permite un aprendizaje estable y eficiente.
- Un learning rate alto puede generar inestabilidad, provocando oscilaciones y dificultando la convergencia del modelo.

En la práctica, los valores intermedios suelen ofrecer mejores resultados.

---

### ¿Cuántas épocas fueron necesarias para la convergencia? ¿De qué depende?

La cantidad de épocas necesarias para que el perceptrón converja depende de diversos factores:

- El grado de separabilidad de los datos.
- El valor del learning rate.
- La inicialización de los pesos y el bias.

En el caso de datos linealmente separables, el perceptrón logra converger. Sin embargo, cuando los datos no son separables, el modelo no alcanza una convergencia completa.
