## 1. Introducción del problema y del dataset

En este trabajo se desarrolló un modelo de clasificación de imágenes capaz de distinguir entre perros y gatos utilizando técnicas de Deep Learning (aprendizaje profundo, una rama avanzada de la inteligencia artificial). Este tipo de problemas pertenece al área de Visión por Computadora (disciplina que permite a las máquinas interpretar imágenes) y tiene múltiples aplicaciones en la vida real, como sistemas de reconocimiento automático, clasificación de contenido y automatización de procesos.

Para este proyecto se utilizó el dataset público **Cats vs Dogs**, disponible a través de TensorFlow Datasets. Un dataset es un conjunto organizado de datos, en este caso, miles de imágenes etiquetadas como perro o gato. Esto lo convierte en una excelente opción para entrenar y evaluar modelos de clasificación binaria (problemas donde solo existen dos posibles clases).

El objetivo principal fue comparar el rendimiento de dos enfoques diferentes:

- Una red neuronal convolucional (CNN, un tipo de red especializada en procesar imágenes) construida desde cero.
- Un modelo de Transfer Learning (aprendizaje por transferencia, reutilización de conocimientos previos) utilizando MobileNetV2 preentrenado.

---

## 2. Carga y exploración inicial

Se cargó el dataset utilizando la librería TensorFlow Datasets, lo que permitió descargarlo automáticamente y acceder a él de manera sencilla.

El conjunto contiene dos clases:

- **Cat**
- **Dog**

Posteriormente, se visualizaron varias imágenes de ejemplo para verificar que los datos se cargaron correctamente y para observar la variedad de poses, colores, tamaños y fondos presentes en las imágenes.

Esta exploración inicial resultó fundamental para comprender la complejidad del problema.

---

## 3. Preprocesamiento y partición de datos

Antes de entrenar los modelos, fue necesario realizar una serie de transformaciones sobre las imágenes:

- Redimensionarlas a un tamaño uniforme de 128x128 píxeles.
- Normalizar los valores de los píxeles al rango [0,1].
- Dividir el dataset en conjuntos de entrenamiento y prueba.

La partición utilizada fue:

- **80% para entrenamiento**
- **20% para prueba**

Además, se aplicó Data Augmentation (técnica que genera variaciones artificiales de las imágenes, como rotaciones o reflejos) durante el entrenamiento para mejorar la capacidad de generalización del modelo y reducir el riesgo de overfitting (cuando el modelo memoriza los datos en lugar de aprender patrones generales).

---

## 4. Entrenamiento del Modelo A (CNN Base)

Como primer enfoque, se implementó una red neuronal convolucional desde cero.

La arquitectura incluyó:

- Capas convolucionales (detectan características visuales como bordes y formas).
- Capas de max pooling (reducen el tamaño de la información manteniendo lo importante).
- Capas densas (realizan la clasificación final).
- Dropout (técnica que desactiva neuronas aleatoriamente para evitar sobreajuste).

Este modelo permitió establecer una línea base sobre la cual comparar el rendimiento de técnicas más avanzadas.

---

## 5. Entrenamiento del Modelo B (Transfer Learning)

Como segundo enfoque, se utilizó MobileNetV2, una red neuronal preentrenada sobre ImageNet (una enorme base de datos de imágenes).

Se congelaron las capas convolucionales del modelo base y se agregaron nuevas capas densas para adaptarlo al problema específico de clasificación entre perros y gatos.

Este método permitió aprovechar características visuales ya aprendidas previamente, reduciendo el tiempo de entrenamiento y mejorando la precisión.

---

## 6. Gráficas de entrenamiento

Se generaron gráficas de:

- Accuracy (porcentaje de aciertos) por época.
- Loss (error del modelo) por época.

Estas visualizaciones permitieron analizar el comportamiento de ambos modelos durante el entrenamiento y detectar posibles problemas de aprendizaje u overfitting.

---

## 7. Evaluación comparativa en test

Para evaluar el rendimiento de ambos modelos, se calcularon las siguientes métricas:

- Accuracy (precisión general).
- Precision (qué tan correctas son las predicciones positivas).
- Recall (capacidad de detectar correctamente los casos positivos).
- F1-Score (promedio equilibrado entre Precision y Recall).

Además, se construyó una matriz de confusión, que permite visualizar los aciertos y errores de clasificación.

Los resultados mostraron una clara superioridad del modelo basado en Transfer Learning.

---

## 8. Análisis de errores

Se analizaron algunas imágenes clasificadas incorrectamente por el modelo.

La mayoría de los errores ocurrieron en imágenes con:

- Fondos complejos.
- Baja iluminación.
- Animales en posiciones poco habituales.
- Primeros planos difíciles de interpretar.

Esto permitió comprender mejor las limitaciones del modelo.

---

## 9. Conclusiones finales

Luego de comparar ambos enfoques, se concluye que el modelo de Transfer Learning obtuvo un rendimiento significativamente superior al de la CNN base.

Esto demuestra la enorme utilidad de reutilizar modelos previamente entrenados, especialmente cuando se trabaja con datasets de tamaño moderado.

Además, el uso de Data Augmentation ayudó a mejorar la capacidad de generalización y a reducir el overfitting.

En definitiva, MobileNetV2 resultó ser la mejor alternativa para este problema de clasificación.

# Glosario

- **Dataset:** conjunto organizado de datos.
- **Deep Learning:** rama de la inteligencia artificial basada en redes neuronales complejas.
- **CNN:** red neuronal especializada en el procesamiento de imágenes.
- **Transfer Learning:** técnica que reutiliza un modelo ya entrenado para una nueva tarea.
- **Data Augmentation:** generación artificial de nuevas imágenes a partir de las originales.
- **Overfitting:** cuando el modelo aprende demasiado los datos de entrenamiento y falla con datos nuevos.
- **Accuracy:** porcentaje total de predicciones correctas.
- **Precision:** exactitud de las predicciones positivas.
- **Recall:** capacidad para encontrar todos los casos positivos.
- **F1-Score:** balance entre Precision y Recall.
- **Matriz de Confusión:** tabla que muestra aciertos y errores del modelo.
- **Época:** una pasada completa del modelo por todos los datos de entrenamiento.
- **Inferencia:** proceso de realizar una predicción con el modelo ya entrenado.
- **Robustez:** capacidad del modelo para funcionar correctamente en distintas condiciones.
