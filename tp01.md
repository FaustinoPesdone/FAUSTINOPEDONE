## Aplicación de Nuevas Tecnologías  
### Machine Learning en productos y servicios reales

---

## Introducción

El Machine Learning (ML) es una rama de la inteligencia artificial que permite a los sistemas aprender a partir de datos y mejorar su rendimiento sin ser programados explícitamente para cada tarea.

Actualmente, muchas aplicaciones y plataformas digitales utilizan Machine Learning para analizar grandes volúmenes de información, identificar patrones y automatizar decisiones. Esto permite mejorar la experiencia de los usuarios, optimizar procesos y aumentar la seguridad en distintos servicios tecnológicos.

En este trabajo se presentan cinco productos o servicios reales que utilizan Machine Learning, explicando qué problema resuelven, qué tipo de aprendizaje automático utilizan y qué datos probablemente necesitan para funcionar.

---

## 1. Google Photos

Sitio web: https://photos.google.com/

Problema que resuelve con Machine Learning

Google Photos permite organizar automáticamente grandes colecciones de fotografías detectando personas, objetos y lugares sin que el usuario tenga que etiquetar manualmente cada imagen.

#### Tipo de Machine Learning

Aprendizaje supervisado.

Justificación

El sistema se entrena con millones de imágenes previamente etiquetadas. A partir de estos ejemplos aprende a reconocer patrones visuales y puede identificar esos mismos objetos en nuevas fotografías.

Datos que probablemente utiliza

- Fotografías subidas por los usuarios  
- Imágenes etiquetadas para entrenamiento  
- Metadatos de las fotos (fecha y ubicación)  
- Información visual como colores y formas

## 2. Amazon

Sitio web: https://www.amazon.com/

Problema que resuelve con Machine Learning

Amazon utiliza Machine Learning para predecir qué productos tendrán mayor demanda. Esto permite optimizar el inventario y mejorar los tiempos de entrega.

#### Tipo de Machine Learning

Aprendizaje supervisado.

Justificación

Los modelos analizan datos históricos de ventas para identificar patrones y predecir futuras compras o tendencias de consumo.

Datos que probablemente utiliza

- Historial de ventas  
- Ubicación de compradores  
- Fechas y horarios de compra  
- Tendencias de consumo  
- Eventos comerciales  

---

## 3. TikTok

Sitio web: https://www.tiktok.com/

Problema que resuelve con Machine Learning

El algoritmo de TikTok selecciona los videos que aparecen en la sección “Para ti”, mostrando contenido personalizado para cada usuario según sus intereses.

#### Tipo de Machine Learning

Aprendizaje supervisado y no supervisado.

Justificación

El sistema analiza el comportamiento del usuario (likes, tiempo de visualización y comentarios) para identificar patrones de interés y recomendar contenido similar.

Datos que probablemente utiliza

- Videos vistos  
- Tiempo de visualización  
- Likes y comentarios  
- Cuentas seguidas  
- Interacciones con el contenido  

---

## 4. Uber

Sitio web: https://www.uber.com/

Problema que resuelve con Machine Learning

Uber utiliza Machine Learning para predecir la demanda de viajes en diferentes zonas y horarios. Esto permite ajustar los precios dinámicamente y mejorar la disponibilidad de conductores.

#### Tipo de Machine Learning

Aprendizaje supervisado.

Justificación

El sistema analiza grandes volúmenes de datos históricos de viajes para identificar patrones y predecir cuándo aumentará la demanda de transporte.

Datos que probablemente utiliza

- Ubicación de usuarios  
- Historial de viajes  
- Hora y día  
- Condiciones climáticas  
- Conductores disponibles  

---

## 5. Ualá (empresa argentina)

Sitio web: https://www.uala.com.ar/

Problema que resuelve con Machine Learning

Ualá utiliza Machine Learning para detectar transacciones sospechosas o fraudulentas en tiempo real y así proteger a los usuarios frente a posibles estafas.

#### Tipo de Machine Learning

Aprendizaje supervisado.

Justificación

Los modelos se entrenan con ejemplos de transacciones clasificadas como fraudulentas o legítimas, permitiendo identificar patrones de fraude en nuevas operaciones.

Datos que probablemente utiliza

- Historial de transacciones  
- Ubicación de compras  
- Tipo de comercio  
- Frecuencia de uso de la tarjeta  
- Dispositivo utilizado  

---

## Conclusión

El Machine Learning se ha convertido en una tecnología fundamental en numerosos servicios digitales. Plataformas de redes sociales, comercio electrónico, transporte y servicios financieros utilizan algoritmos de aprendizaje automático para analizar datos y mejorar sus sistemas.

Los ejemplos analizados demuestran cómo esta tecnología permite automatizar procesos, personalizar la experiencia de los usuarios y aumentar la seguridad en diferentes aplicaciones.
